À quoi sert le FluxoScript ?

Comment est interprété le FluxoScript ?

Le Fluxoscript intéroge notre base de donnée pour compiler. Les plus gros avantage de FluxoScript est à quel point il est intuitif et léger. En effet, étant deux étudiants en informatiques, on ne pouvait pas proposer un langage informatique comme Java, c'est pourquoi le FluxoScript sera abordable pour ceux qui débutent en informatique. De plus, ce langage est léger, c'est à dire qu'il ne demande que quelques caractères pour ensuite être interprêté, l'objectif étant de proposer un code dynamique, tout en étant un code qui ne ralentis pas la page. Par exemple, pour vous mettre l'eau à la bouche, un exemple de code en FluxoScript :

fct(var): if var === 1 : return True else False;

Intervient t-il Avant/Après le traitement server ?

Étants deux jeunes développeurs en informatique, nous avons longtemps cotoyer Php, assez pour se dire qu'il faudrait un langage équivalent au Php mais qui interviendrait après le traitement du server, afin de proposer à ces adeptes de nouvelles fonctionnalités.

Quel est l'avantage d'une intervention après traitement server ?

C'est vraiment dommage de générer de manière dynamique du code et qu'il ne soit pas interprêté, c'est pourquoi le FluxoScript vient résoudre vos problèmes ! Imaginez appeler de manire dynamique du code qui est ensuite interprêté, vous pouvez donc une fois maîtrisé, utiliser le Fluxoscript pour dynamiser une page web, tout en appelant du Fluxoscript de manière dynamique, grâce au FluxoScript.

Vous commencez à comprendre, vous devez au plus vite apprendre le FluxoScript afin de maîtriser cet outils avant tout le monde !
