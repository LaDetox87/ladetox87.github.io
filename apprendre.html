La syntaxe


Pour appeler une fonction :
fct(param,param2):

Les return :
rtn True;

Les if :
if param === 1 : rtn True else False;

Voici un Équivalent :
if param === 1 : rtn True;
else if param === 2: rtn False;
else : rtn None;

les Expressions :
affectation de valeur : =
test d'égalité de valeur : ==
test d'égalité de valeur et de type : ===
Différence de valeur : !=
différence de valeur et type : !==
pour inverser une expression, None(True); -> False

La Typologie :
Integer : int
String : str
Float : flt
Liste : lst -> lst liste1 = [param,param2];
liste1.add(param);
liste1.rmv(param2);
Char -> il n'en existe pas en FluxoScript, un char n'est qu'un String de longueur 1.

les fonctions utilisables :
len("aaa");
